Full Website

Website Github

O projeto desse site se deu por uma necessidade de atualização do site da empresa, o proprietário e grande amigo Rafael Piga da Marmoraria União, esta a mais de 25 anos no ramo junto de seu pai, sendo a segunda geração no negócio. Aprender mais sobre a empresa dele, o produto, que quando trabalhadas por profissionais que amam oque são verdadeiras obras de arte foi incrível.

Lidar com um cliente assim acaba trazendo uma energia extra para a concepção e entrega do projeto, faz você superar qualquer barreira para dar o seu melhor, não poderia entregar menos do que ele merece.

O site conta com uma quantidade grande de imagens e utilizar o GatsbyJS com GraphQL foi uma decisão muito assertiva nesse aspecto, outro ponto que contou muito foi a velocidade, e que velocidade veja aqui todo o poder do ReactJS correndo por trás do GatsbyJS fazendo dele um PWA (Progressive Web App) potente. Para mais detalhes sobre oque foi utilizado, deixei o repositório do Github aberto, fiquem a vontade.
