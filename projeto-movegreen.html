Website Github

A MoveGreen nasceu para atender uma demanda cada vez maior dos brasileiros; veículos elétricos de mobilidade urbana que podem ser facilmente transportados.

Feito em ReactJS e NodeJS, o repositório do Github esta aberto para outros Devs que queiram contribuir ou ver quais ferramentas foram utilizadas :)

No momento a loja-online encontra-se em desenvolvimento, porem caso queira testa-la, um numero de cartao de credito ficticio sera fornecido ao longo da compra, e seu cadastro vai ficar no nosso banco de dados, para em breve lhe atender.
